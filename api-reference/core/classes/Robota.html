Core API - v0.1.0 / Robota

Class: Robota

Robota의 메인 클래스 에이전트를 초기화하고 실행하는 인터페이스 제공

Example

const robota = new Robota({
  provider: new OpenAIProvider({
    model: 'gpt-4',
    client: openaiClient
  }),
  systemPrompt: '당신은 도움이 되는 AI 어시스턴트입니다.'
});

const response = await robota.run('안녕하세요!');
